## Dataset Descriptions

### Titanic Dataset

Contains passenger information from the Titanic disaster, including survival status, class, gender, age, and other features.

### Sonar Rock vs Mine Dataset

Contains sonar signals bounced off either rocks or metal cylinders (mines), used for binary classification.

### COVID-19 Dataset

Contains COVID-19 related data including cases, deaths, and other relevant information.

### Diabetes Dataset

Contains medical data for diabetes prediction/classification tasks.

### Churn Modelling Dataset

Customer data for churn prediction in a banking context.

### Chai Sales Dataset

Sales data for a product called "Chai".

### Breast Cancer Classification Dataset

Medical data for breast cancer classification.

### Movies Recommendation Dataset

Movie ratings and metadata for building recommendation systems.

### Spam Mail Detection Dataset

Email data for spam/ham classification.

### Exercise Dataset

Exercise activity data for analytics or prediction.

### Calories Burnt Dataset

Data on calories burnt during various activities.

### Customer Segmentation Dataset

Customer data for segmentation and clustering tasks.

### BigMart Sales Dataset

Retail sales data from BigMart stores.

### Medical Insurance Dataset

Insurance data for cost prediction or analysis.

### Heart Disease Dataset

Medical data for heart disease prediction/classification.

### Gold Price Dataset

Historical gold price data for forecasting or analysis.

### Car Price Dataset

Car features and prices for regression or analysis.

### Loan Prediction Dataset

Loan application data for approval prediction.
